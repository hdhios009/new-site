# IQ — школа развития интеллекта

Сайт детской школы развития интеллекта «IQ» (Петропавловск-Камчатский).

## Направления
- Подготовка к школе
- Скорочтение
- Развитие интеллекта
- Каллиграфия

## Структура
```
iq-site/
├── index.html    — главная страница
├── policy.html   — политика конфиденциальности
├── images/       — фотографии центра
└── README.md
```

Страницы `index.html` и `policy.html` полностью автономны (работают офлайн, без интернета и установки). Папка `images/` содержит исходные фотографии.

## Публикация на GitHub Pages
1. Загрузите содержимое этой папки в репозиторий.
2. Settings → Pages → Deploy from branch → `main` / root.
3. Сайт откроется по адресу `username.github.io/имя-репозитория`.
