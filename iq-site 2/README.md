# IQ — школа развития интеллекта

Сайт детского образовательного центра «IQ» (Петропавловск-Камчатский).

## Направления
- Подготовка к школе
- Скорочтение
- Развитие интеллекта

## Структура
```
iq-site/
├── index.html      — главная страница (полностью автономная)
├── images/         — фотографии центра
└── README.md
```

## Как открыть
Откройте `index.html` в любом браузере — сайт работает без интернета и без установки.

## Публикация на GitHub Pages
1. Загрузите содержимое папки в репозиторий.
2. Settings → Pages → Deploy from branch → `main` / root.
3. Сайт откроется по адресу `username.github.io/имя-репозитория`.
